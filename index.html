This repository juj/audio_test_suite contains a set of audio test files for different audio related effects.
  • 8bit_detective: Tests seamlessly looping playback of a highly compressed 8kHz quality mono music clip.
  • shanghai_action: Tests seamlessly looping playback of a low sample rate 16kHz mono music clip.
  • jungle_loop: Tests seamlessly looping playback of a good quality 44.1kHz stereo music clip.
  • biomechanics: Tests seamlessly looping playback of a good quality 44.1kHz music clip with an intro and custom loop points.
  • catedral: Tests animating pitch of a music file.
  • car_acceleration_high: Tests animating pitch while seamlessly looping an audio clip.
  • cinematic: Tests seamlessly playing back a looping clip with intro, variable amount of loops, and an outro.
  • road_trip: Tests 2D stereo panning via a 3D spatialized sound source.
  • bell_trip: Tests implementing a doppler effect on a fast moving looping 3D sound source.
